# しらないタイピング (Siranai Typing)

WindowsのIMEにある「知らない打ち方」を楽しむ、タイピングゲームです。
「より少ない打鍵数」で入力できるかを試すことができます。

## 特徴 (Features)

* **柔軟な入力判定ロジック:**
    * **最適化入力:** `shi` (3打) だけでなく `si` (2打) や `ci` (2打) も正解として受け付けます。
    * **拗音 (Compound Kana):** `kya` のような2文字同時消費と、`ki` `ya` のような単独入力を両立。
    * **促音 (Sokuon):** `tt` のような子音重ね打ちと、`xtu` のような単独入力に対応。
* **漢字・読み仮名対応:** CSV形式のデータにより、漢字の表示と読み仮名の判定を分離。
* **リアルタイムフィードバック:** 入力中のローマ字を可視化。
* **計測機能:** タイム計測、総タイプ数、ミスタイプ数のカウント。
